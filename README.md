# 📍 Dishanirya – AR Campus Navigation System

An interactive web-based campus navigation system that helps students and visitors explore campus facilities through an intuitive interface and AR-inspired navigation. The application provides building information, floor-wise navigation, an interactive campus map, and a searchable directory for easy navigation. The project is built using HTML, CSS, and JavaScript. 

---

## 🚀 Features

- 🗺️ Interactive campus map
- 📍 Building directory with search functionality
- 🏫 Detailed pages for campus facilities
- 🧭 Floor-wise navigation (2nd & 3rd floors)
- 📱 Responsive and user-friendly interface
- 🎥 AR-inspired navigation interface using the device camera
- 📍 Google Maps integration for navigation
- 🔍 Quick access to campus departments and facilities

## ▶️ How to Run

1. Download or clone this repository.

```bash
git clone https://github.com/manasvikorutla-ui/Dishanirya.git
```

2. Open the project folder.

3. Open **Dishanirya.html** in your web browser.

4. Allow camera permission (if prompted) to experience the AR-style navigation.
